# Simulateur de Capacité de Giratoire (Basé sur Girabase)

Cette application Streamlit permet d'évaluer la réserve capacitaire et le taux de charge d'un carrefour à sens unique (giratoire) selon les méthodes de calcul du CEREMA.

## Installation locale
1. Cloner le repo.
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer l'app : `streamlit run streamlit_app.py`
